Contribute to Roole

Donation

Roole has been helpful? Please consider donating to help me keep improving Roole and work on more open source projects like Roole. :)

Donate with: PayPal, Gittip, Flattr

Since PayPal doesn't allow Asia regions (I live in China) to receive donations, please manually send donation to my PayPal account curvedmark@gmail.com.

Improving the Website

Improve the website by editing the source files of this website on GitHub. You can edit these files directly in the browser by clicking the "edit" button for each files on GitHub.

Releasing Libraries

You can make contributes to the Roole ecosystem by releasing awesome Roole libraries (e.g., utility libraries, UI libraries, etc). So users can benifit from your works and more people will be drawn to Roole.

Sending Feedbacks

If the language contains bugs, or doesn't work as you intended, don't abandon it immediately, open an issue on the issue tracker.

If you got a great idea for new features, don't hesitate to open a new issue too.

Spreading the Word

You can help Roole by telling more people about it, write tutorials on how to use it, compare it with other CSS preprocessors, etc.
